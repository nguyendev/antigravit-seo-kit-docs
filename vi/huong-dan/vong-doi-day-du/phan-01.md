# 1: Onboard — Khởi Tạo Dự Án

Bắt đầu bằng `/seo-run yourdomain.com`. SEO Kit phát hiện dự án mới và gợi ý `/seo-onboard`. Chấp nhận để khởi tạo.

## Quá trình

1. Tạo workspace tại `seo-projects/yourdomain-com/`
2. Kiểm tra Google API credentials
3. Phân tích brand identity (Source Context)
4. Xây dựng audience personas
5. Phát hiện ngành và đối thủ

## Kết quả

- `project.json` với phase: "onboarded"
- Báo cáo onboarding với brand profile, API readiness, industry type
