# Vòng Đời Đầy Đủ (5 Phase)

Trải nghiệm toàn bộ vòng đời SEO từ onboarding đến monitoring.

## Phase 1: Onboard

```
/seo-onboard yourdomain.com
```

## Phase 2: Research + Audit

```
/seo-research yourdomain.com --focus all
/seo-audit yourdomain.com --scope priority
```

## Phase 3: Strategy

```
/seo-strategy yourdomain.com competitor1.com competitor2.com
```

## Phase 4: Execute

```
/seo-execute yourdomain.com
```

## Phase 5: Monitor

```
/seo-monitor yourdomain.com
```

> 💡 Sau Phase 5, bạn có thể quay lại Phase 2 (re-audit) để lặp lại chu kỳ.
