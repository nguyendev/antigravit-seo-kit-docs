# seo-keyword — Từ Khóa

Nghiên cứu từ khóa toàn diện: seed expansion, volume/difficulty, intent classification, clustering.

## Chức năng
- Mở rộng seed keywords → universe 500+ keywords
- Phân loại search intent (informational, commercial, transactional, navigational)
- Clustering theo topic
- Scoring theo opportunity (volume × intent / difficulty)
- Output: keywords JSON, clusters JSON
