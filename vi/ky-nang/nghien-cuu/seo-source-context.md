# seo-source-context — Ngữ Cảnh Nguồn (DNA Context)

## Mô Tả

Phân tích và xây dựng Source Context (DNA Context): semantic anchor, authorship, brand identity, trust signals, about page, credentials.

## Agent Phụ Trách

`seo-technical-architect` (checking) + `seo-creator` (building) — **shared skill**

## Chức Năng Chính

- **Semantic anchor**: Xác định core topic authority của website
- **Authorship analysis**: Kiểm tra author bios, credentials, expertise
- **Brand identity**: Đánh giá brand signals trên website
- **Trust signals**: About page, contact info, testimonials, certifications
- **DNA alignment scoring**: Multi-signal engine (unigram, bigram, topical map, substring)
- **Topical authority map**: Silo structure + thematic coverage
