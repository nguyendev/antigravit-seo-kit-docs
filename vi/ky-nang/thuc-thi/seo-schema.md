# seo-schema — Schema Markup

Detection, validation, và generation Schema.org markup (JSON-LD, Microdata, RDFa).

## Chức năng
- Detect existing schema trên trang
- Validate against Google's rich result types
- Check deprecated schemas (HowTo, FAQ, SpecialAnnouncement)
- Generate missing schema recommendations
- Schema.org v29.4 support
- Industry-specific schema templates
