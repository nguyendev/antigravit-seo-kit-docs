# seo-migration — Di Chuyển Website

## Mô Tả

Quản lý di chuyển website: domain migration, CMS migration, URL structure changes, redirect mapping, post-migration monitoring.

## Agent Phụ Trách

`seo-technical-architect` (skill #1/2)

## Chức Năng Chính

- **Redirect mapping**: URL-to-URL redirect plan
- **Pre-migration audit**: Benchmark current SEO metrics
- **Post-migration check**: Verify redirects, indexing, rankings
- **URL migration**: Map old URLs → new URLs
- **Domain migration**: Old domain → new domain
- **CMS migration**: WordPress → Next.js, etc.

## Khi Nào Dùng

- "site migration", "di chuyển website", "redirect mapping"
- "domain migration", "URL migration", "post-migration"

## Workflow Liên Quan

- `/seo-execute --migration` — Migration trong execute phase
