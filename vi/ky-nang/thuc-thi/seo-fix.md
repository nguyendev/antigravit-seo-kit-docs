# seo-fix — Sửa Lỗi Kỹ Thuật Tự Động

## Mô Tả

Tự động tạo bản sửa lỗi kỹ thuật SEO: meta tags, schema, robots.txt, redirects, code fixes dựa trên kết quả audit.

## Agent Phụ Trách

`seo-migration-expert` (skill #2/2)

## Chức Năng Chính

- **Meta tags fix**: Title, description auto-generation
- **Schema fix**: Tạo/sửa JSON-LD markup
- **Robots.txt fix**: Sửa crawl directives
- **Redirect mapping**: 301/302 redirect rules
- **HTML fixes**: Canonical, hreflang, meta robots
- **Bulk generation**: Xử lý nhiều pages cùng lúc

## Khi Nào Dùng

- "sửa lỗi SEO", "auto fix", "meta tags fix"
- "schema fix", "redirect", "technical fix"

- `/seo-execute` — Generate fixes từ audit recommendations
