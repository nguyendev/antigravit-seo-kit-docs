# seo-fix — Tự Động Sửa Lỗi

Auto-generate SEO code fixes: meta tags, schema JSON-LD, robots.txt, redirects.

## Chức năng
- Tạo meta tags tối ưu (unique, keyword-first, compelling CTR)
- Generate schema JSON-LD hoàn chỉnh
- Tạo/update robots.txt với AI crawler rules
- Redirect mapping cho 404s
- Ready-to-use code files
