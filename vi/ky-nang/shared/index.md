# Kỹ Năng Nền Tảng (Shared Domain)

Các kỹ năng nền tảng (Shared Skills) là những khối chức năng được sử dụng chung cho cả mục đích **SEO** lẫn **Web Development**. Chúng đóng vai trò bảo vệ hệ thống, tối ưu hóa quá trình làm việc, và đảm bảo chất lượng đầu ra.

## Các AI Agents Dùng Chung

Hệ thống cung cấp **10 Shared Agents** để hỗ trợ các vòng đời dự án:

| Agent | Nhiệm vụ |
|-------|----------|
| `project-planner` | Lập kế hoạch, chia nhỏ tasks (Work break-down), quản lý dependency graph. |
| `security-auditor` | Quét lỗ hổng OWASP, kiểm tra bảo mật API và mã nguồn. |
| `performance-optimizer` | Tối ưu hóa dung lượng (bundle size), chạy Lighthouse audit. |
| `debugger` | Gỡ lỗi chuyên sâu, phân tích root-cause. |
| `explorer-agent` | Phân tích codebase, tìm kiếm quy tắc và kiến trúc thư mục. |
| `documentation-writer` | Viết tài liệu (như chính trang docs này), thêm code comments. |
| `devops-engineer` | Quản lý Server, Docker, CI/CD pipeline. |
| `qa-automation-engineer` | Thiết lập môi trường và chạy test E2E tự động. |
| `test-engineer` | Viết Unit Test và Integration Test. |
| `code-archaeologist` | Xử lý Legacy Code, đọc hiểu hệ thống cũ để refactor. |

## Các Kỹ Năng Nổi Bật (21 Shared Skills)

- **vulnerability-scanner**: Script tự động quét bảo mật `security_scan.py` chạy trước mỗi lần deploy.
- **systematic-debugging**: Quy trình 4 bước (4-phase) gỡ lỗi hệ thống bài bản.
- **performance-profiling**: Sử dụng `lighthouse_audit.py` và `bundle_analyzer.py` để giữ website luôn xanh (Green Web Vitals).
- **clean-code**: Quy chuẩn viết code sạch, dễ bảo trì, dễ đọc bởi cả người và AI.
- **mcp-builder**: Kỹ năng xây dựng Model Context Protocol server tùy chỉnh cho dự án của bạn.
- **red-team-tactics**: Kỹ thuật tấn công giả lập để phát hiện điểm yếu của hệ thống.
- **plan-writing**: Kỹ thuật viết Kế hoạch thực thi để Orchestrator quản lý.

*(Bao gồm thêm các kỹ năng quản trị Server, Bash/Linux, Powershell/Windows, TDD Workflow...)*

Những kỹ năng này giúp cho quá trình triển khai Antigravity SEO Kit trở thành một cỗ máy tự động hóa chuyên nghiệp ở cấp độ doanh nghiệp (Enterprise-grade).
