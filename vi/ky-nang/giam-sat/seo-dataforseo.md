# seo-dataforseo — DataForSEO API Integration

## Mô Tả

Tích hợp DataForSEO API: live SERP, keyword volume, backlinks, AI visibility data. Provider dữ liệu all-in-one cho SEO Kit.

## Agent Phụ Trách

`seo-intelligence` (skill #2/4)

## Chức Năng Chính

- **Live SERP**: Kết quả tìm kiếm real-time
- **Keyword data**: Volume, CPC, difficulty, trends
- **Backlink API**: Referring domains, anchor text, new/lost
- **AI visibility**: AI mention tracking qua DataForSEO
- **Domain analytics**: Organic traffic estimates
- **22 commands**: Bao phủ hầu hết nhu cầu SEO data

## Yêu Cầu

> Cần **DataForSEO extension** + API key. Xem [DataForSEO Extension](../../mo-rong/dataforseo.md).
