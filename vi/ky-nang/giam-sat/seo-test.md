# seo-test — SEO A/B Testing

## Mô Tả

Thử nghiệm SEO: A/B test titles, meta descriptions, tracking SEO experiments, đo lường impact của thay đổi.

## Agent Phụ Trách

`seo-data-analyst` (skill #4/4)

## Chức Năng Chính

- **Meta description testing**: Test descriptions khác nhau
- **SEO experiment design**: Setup controlled experiments
- **Impact measurement**: Đo lường thay đổi rankings, traffic
- **Statistical significance**: Đảm bảo kết quả đáng tin
