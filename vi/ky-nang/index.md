> 5 skills chia sẻ giữa agents: `seo-content`, `seo-solann`, `seo-google`, `seo-dataforseo`, `seo-information-gain` — cùng tên nhưng khác mục đích tùy agent.

## Cách Sử Dụng

Skills được kích hoạt tự động khi chạy workflows. Bạn cũng có thể gọi trực tiếp:

```
Chạy seo-entity analysis cho example.com
```

Lưu ý: gọi skill trực tiếp **KHÔNG** thay đổi project phase.
