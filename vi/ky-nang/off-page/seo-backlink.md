# seo-backlink — Link Building & Analysis

## Mô Tả

Phân tích và xây dựng backlinks: toxic link detection, anchor text analysis, link gap analysis, referring domains, domain authority metrics.

## Agent Phụ Trách

`seo-growth-hacker` (skill #1/9)

## Chức Năng Chính

- **Backlink profile analysis**: Đánh giá tổng quan link profile
- **Toxic link detection**: Phát hiện links độc hại, spam
- **Anchor text distribution**: Phân tích phân bổ anchor text
- **Link gap analysis**: So sánh backlinks vs competitors
- **Referring domains**: Đánh giá chất lượng domains trỏ về
- **Link building opportunities**: Đề xuất cơ hội xây dựng links
