# Xây Dựng Web Chuẩn SEO (Web Development)

Với bản cập nhật **The Grand Unification**, Antigravity SEO Kit không chỉ là công cụ phân tích mà còn là công cụ **thực thi mã nguồn (code execution)**. Phân hệ Web Development biến các khuyến nghị SEO thành những dòng code cụ thể, xây dựng một hệ sinh thái web tốc độ cao, thân thiện với người dùng và cả AI Agents.

Mục tiêu cốt lõi: **Code để Rank**. Mọi dòng code, cấu trúc React, cấu hình Tailwind hay Database Schema đều hướng tới việc đạt điểm số Core Web Vitals tối đa và chuẩn WebMCP.

## Các AI Agents

| Agent | Chức năng chính |
|-------|-----------------|
| `web-orchestrator` | Điều phối lập trình, phân chia công việc cho Frontend/Backend. |
| `web-frontend-specialist` | Chịu trách nhiệm UI/UX, React Rendering, TailwindCSS, và loại bỏ waterfalls để tối ưu Core Web Vitals. |
| `web-backend-specialist` | Chịu trách nhiệm API Design, cấu trúc Database, và cấu hình WebMCP Discoverability cho AI Agents. |

## Các Kỹ Năng Chính (12 Web Skills)

- **web-app-builder**: Khởi tạo cấu trúc dự án từ các template chuẩn SEO (Astro, Nextjs Static, v.v.).
- **web-frontend-design**: Thiết kế hệ thống UI/UX, màu sắc, animation thân thiện người dùng và đạt chuẩn Accessibility.
- **web-react**: Chuyên sâu tối ưu React (Server Components, loại bỏ waterfalls, thu nhỏ bundle size, giảm re-renders).
- **web-tailwind**: Quản lý CSS chuẩn xác, cấu trúc utility-first với container queries.
- **web-api-patterns**: Thiết kế REST/GraphQL, xác thực, và đảm bảo API discoverability.
- **web-database**: Thiết kế schema tối ưu, indexing nhanh chóng.
- **web-testing**: Viết unit test, E2E test bằng Playwright để đảm bảo UI không lỗi.
- **web-deployment**: Triển khai dự án an toàn với 5-phase rollout.
- *(và các kỹ năng công nghệ khác như web-python, web-nodejs...)*

## Web Workflows

Khác với `/seo-*` chuyên dùng để phân tích và tối ưu, các workflow `/web-*` dùng để lập trình và xây dựng website:

- `/web-create`: Tạo ứng dụng/chức năng mới từ đầu.
- `/web-enhance`: Nâng cấp chức năng có sẵn trên source code.
- `/web-ui-design`: Tạo UI theo chuẩn Design System.
- `/web-test`: Tự động viết test cho các components.
- `/web-preview`: Chạy dev server để xem trước kết quả.
- `/web-deploy`: Build và đưa lên production.
- `/web-debug`: Gỡ lỗi code.

Khi làm việc với các lệnh này, hệ thống sẽ tự động tuân thủ các quy tắc SEO đã được định nghĩa ở lớp nền tảng.
