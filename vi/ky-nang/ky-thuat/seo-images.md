# seo-images — Hình Ảnh

Phân tích tối ưu hình ảnh: alt text, file sizes, formats, lazy loading, CLS prevention. Kích hoạt trong `/seo-audit`.

## Chức năng
- Kiểm tra alt text cho tất cả images
- Phân tích file sizes và đề xuất nén
- Kiểm tra format hiện đại (WebP, AVIF)
- Validate lazy loading implementation
- Phát hiện CLS do images gây ra (missing width/height)
