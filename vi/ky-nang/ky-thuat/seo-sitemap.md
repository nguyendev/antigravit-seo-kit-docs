# seo-sitemap — Sitemap

Phân tích XML sitemap: coverage, quality, industry templates. Kích hoạt trong `/seo-audit`.

## Chức năng
- Kiểm tra sitemap.xml tồn tại và khả dụng
- Phân tích coverage (URLs trong sitemap vs URLs thực tế)
- Kiểm tra lastmod, changefreq, priority
- Validate cấu trúc sitemap index
- Đề xuất theo industry template
