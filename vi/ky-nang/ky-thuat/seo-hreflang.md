# seo-hreflang — Đa Ngôn Ngữ

Validate hreflang tags cho SEO đa ngôn ngữ. Kích hoạt trong `/seo-audit`.

## Chức năng
- Kiểm tra hreflang implementation (HTML tags, HTTP headers, sitemap)
- Validate return links (bidirectional)
- Kiểm tra x-default tag
- Phát hiện conflicting signals
- Đề xuất cải thiện cho multi-language sites
