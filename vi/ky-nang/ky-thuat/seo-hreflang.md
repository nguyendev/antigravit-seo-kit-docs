# seo-hreflang — Đa Ngôn Ngữ / International SEO

## Mô Tả

Kiểm tra và tối ưu hreflang tags cho website đa ngôn ngữ: mapping locale, return tags, x-default, conflicts.

## Agent Phụ Trách

`seo-auditor` (skill #8/10)

## Chức Năng Chính

- Kiểm tra hreflang implementation (HTML, HTTP header, sitemap)
- Phát hiện missing return tags
- Validate locale codes (ISO 639-1 + ISO 3166-1)
- x-default analysis
- Conflict detection (multiple hreflang cho cùng locale)
- Self-referencing tag verification
