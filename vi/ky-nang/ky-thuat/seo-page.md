# seo-page — Phân Tích Trang

On-page SEO analysis cho một URL cụ thể. Kích hoạt trong `/seo-page`.

## Chức năng
- Title tag, meta description analysis
- Heading hierarchy (H1-H6)
- Internal link structure, anchor text
- URL structure evaluation
- Canonical, meta robots, Open Graph, Twitter Card
- Keyword usage và placement
