# seo-page — Phân Tích Trang Đơn

## Mô Tả

Phân tích on-page SEO cho một URL cụ thể: title, meta, headings, content quality, schema, internal links.

## Agent Phụ Trách

`seo-creator` (skill #3/7)

## Chức Năng Chính

- **Title tag**: Length, keyword placement, uniqueness
- **Meta description**: Length, CTR optimization
- **Headings**: H1 uniqueness, heading hierarchy
- **Content**: Word count, readability, keyword density
- **Schema markup**: Validation, missing types
- **Internal links**: Anchor text, link depth
- **Images**: Alt text, lazy loading
- **URL structure**: Length, keywords, slugs
