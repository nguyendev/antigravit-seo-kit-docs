# Bắt Đầu

Antigravity SEO Kit là bộ công cụ phân tích SEO chuyên nghiệp gồm **35 kỹ năng chuyên biệt**, được tổ chức thành **vòng đời 5 giai đoạn** cho nền tảng [Google Antigravity](https://deepmind.google/technologies/antigravity/).

## Cài Đặt Nhanh

```bash
npx ag-seo-kit install --key=SK-XXXX-XXXX-XXXX
```

> Mua license key tại: [https://antigravity-seokit.solann.io/](https://antigravity-seokit.solann.io/)

## Bước Tiếp Theo

Sau khi cài đặt thành công:

1. Đọc [Yêu cầu hệ thống](yeu-cau-he-thong.md) để đảm bảo môi trường đầy đủ
2. Xem [Cài đặt chi tiết](cai-dat.md) nếu cần cấu hình nâng cao
3. Thực hành [Audit đầu tiên](audit-dau-tien.md) để làm quen với hệ thống
4. Tìm hiểu [Quản lý license](quan-ly-license.md) để quản lý thiết bị
