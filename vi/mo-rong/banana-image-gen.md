# Banana Image Gen Extension

AI-generated SEO image assets qua Gemini AI: OG images, hero images, product photos, infographics.

## 6 Lệnh

| Lệnh | Mô tả |
|------|-------|
| `generate_og_image` | Open Graph image cho social sharing |
| `generate_hero_image` | Hero section image |
| `generate_product_image` | Product photo generation |
| `generate_infographic` | Infographic layout |
| `generate_blog_image` | Blog post featured image |
| `generate_icon_set` | Icon set cho website |

## Cấu Hình

Yêu cầu Banana.dev API key và Gemini model access.
