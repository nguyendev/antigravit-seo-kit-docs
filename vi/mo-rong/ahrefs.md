# Ahrefs Extension

Cung cấp dữ liệu backlink chất lượng cao, DR/UR metrics, content gap analysis.

## Chức năng
- Domain Rating (DR), URL Rating (UR)
- Backlink quality analysis
- Content gap vs competitors
- Referring domains tracking
- Top pages analysis

## Cấu Hình

Yêu cầu Ahrefs API key. Xem hướng dẫn tại [ahrefs.com/api](https://ahrefs.com/api).
